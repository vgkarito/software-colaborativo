# Software Colaborativo

Repositorio creado para la materia de **Software Colaborativo** como parte de la carrera de Ingeniería en Inteligencia Artificial en Hybridge.

## 📌 Descripción

Este repositorio documenta los temas, actividades y proyectos desarrollados a lo largo del curso, con enfoque en buenas prácticas de desarrollo colaborativo, uso de herramientas de control de versiones y trabajo en equipo.

## 📚 Temas del curso

- Open Source y sus enfoques (FSF, OSI, Open Ecosystem)
- Tipos de licencias de software (MIT, Apache, GNU GPL)
- Control de versiones con Git y GitHub
- Flujos de trabajo colaborativo

## 🛠️ Tecnologías utilizadas

- Git
- GitHub
- Python
